学习 Flutter 时写的 Demo 项目.
 
## 如何使用
```bash
git clone https://github.com/liuzhiyuan1993/hello_flutter
cd hello_flutter

# 1. 需要已安装 Flutter
# 2. 需要安装手机模拟器, 例如 Mac 下的 Simulator
flutter run
```
